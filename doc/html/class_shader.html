Shader::Shader ( const std::string &  shaderDir,
LoadFlag  loadFlags 
)

Load a Shader from directory with the necessary files. The following naming convention is assumed: vertex shader: vert.glsl fragment shader: frag.glsl geometry shader: geom.glsl tesseval shader: eval.glsl tesscontrol shader: cont.glsl

These flags are used to specify the shader type: LOAD_BASIC : load fragment + vertex shader (always the case) LOAD_GEOM : load geometry shader LOAD_TESS : load tesselation shader LOAD_FULL = LOAD_GEOM | LOAD_TESS : load tesselation and geometry shader

Parameters
shaderDirdirectory where the shader files are located
loadFlagsspecifies what kind of shaders should be loaded
